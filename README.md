# termux-settings

## Usage

For init only!

```
export ACTION=init
```

For configure

```
export SSH_PORT=1082
```

```
export USER_PASSWORD=somepass
```

```
curl -o ~/init.sh https://raw.githubusercontent.com/EvgeniyBlinov/termux-settings/master/init.sh && chmod +x ~/init.sh && bash ~/init.sh
```
